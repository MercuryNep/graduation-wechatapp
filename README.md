学生活动学时统计小程序


项目简介
本项目基于微信开发者工具 + 腾讯云数据库开发，专为在校学生设计，用于精准统计并管理参与各类社会活动所获得的学时。学生可通过小程序一键报名、签到、实时查看学时余额；教师/管理员可在后台批量审核、导出报表，彻底告别手工Excel统计，让学时管理更高效、透明、可追溯。

核心功能
角色	功能
学生	活动浏览、一键报名、扫码签到、学时明细、年度进度条
教师	活动发布、签到二维码、学时审批、补录申请、数据导出
管理员	用户管理、异常学时预警
技术栈
前端：微信小程序原生框架 + ColorUI + WeUI
后端：云开发（CloudBase）+ 云函数（Node.js 16）
数据库：腾讯云云数据库（MongoDB 4.4）
存储：云存储（活动海报、证书PDF）
运维：云监控、日志服务、CI/CD（微信开发者工具·云托管）
