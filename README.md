学生活动学时统计小程序


项目简介
本项目基于微信开发者工具 + 腾讯云数据库开发，专为在校学生设计，用于精准统计并管理参与各类社会活动所获得的学时。学生可通过小程序一键报名、签到、实时查看学时余额；教师/管理员可在后台批量审核、导出报表，彻底告别手工Excel统计，让学时管理更高效、透明、可追溯。
